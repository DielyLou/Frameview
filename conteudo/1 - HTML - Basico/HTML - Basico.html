Tags

As tags servem para demarcar os tipos de conteúdo dentro de uma página, e tambem, para que os mecanismos de busca possam identificar onde se localizará cada tipo de conteúdo na pagina.


As tags precisam ser abertas e fechadas para o navegador entender onde um contúdo começa e onde ele termina.


A união de uma tag de abertura (com um conteúdo) e uma tag de fechamento é denominada de elemento. Existem tags que se autocompletam, então não é necessario um fechamento.

Títulos e parágrafos:

As tags de títulos são muito importantes no site, sendo enumeradas de 1 a 6, dessa maneira:

  1. h1.
  2. h2.
  3. h3.
  4. h4.
  5. h5.
  6. h6.

A tag “h1” é a que tem mais importância e a “h6” a que tem menos importância. É melhor que se tenha somente um único “h1” na página.


As tags de parágrafo funcionam para fazer parágrafos e dar espaço entre eles, dessa forma, o texto fica mais bem organizado.

Tags de formatação

Essas tags formatam textos ou palavras que vão fazer parte do conteúdo da página. É melhor que se opte por sempre usar tags semânticas, por conta dos mecanismos de busca, e somente usar as tags não semânticas em casos bem específicos. Esses são exemplos de tags de formatação:


  • Para deixar o texto em negrito, pode-se usar a tag “(b)” e a tag “(Strong)”, mas semanticamente, usar a tag “Strong” é melhor, já que ela auxilia na acessibilidade de deficientes.
  • Para deixar o texto ou uma palavra em itálico, se usam as tags “i” e a tag “em”, porém, pelo mesmo motivo das anteriores, a tag “em”, é semanticamente melhor.
  • Para deixar o texto ou palavra com uma marcação, se usa a tag “mark”.
  • Para deixar o texto ou palavra pequeno, se usa a tag “small”.
  • Para deixar um texto ou palavra com um traço atravessado, se usam as tags “s” e a tag “del”, mas semanticamente usar a tag “dell” é melhor.
  • Para deixar um texto ou palavra com um risco de sublinhado, se usam as tags “u” e a tag “ins”, mas semanticamente a tag “ins” é melhor.
  • Para deixar um texto ou palavra subscrito, se usa a tag “sub”, muito utilizada em artigos científicos.
  • Para deixar um texto superescrito, se usa a tag “sup”, muito usada para escrever contas de matemática.
